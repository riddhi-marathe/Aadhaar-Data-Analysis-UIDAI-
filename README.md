# Aadhaar Intelligence & Analytics System

An interactive analytics platform for analyzing Aadhaar enrolment,
biometric, and demographic datasets using visualization, anomaly
detection, and predictive modeling.

## Features
- Interactive dashboards
- Anomaly detection
- Trend forecasting
- PDF reports

## How to Run
pip install -r requirements.txt
streamlit run app.py

## Note
Datasets are excluded for privacy reasons.
